Objectifs du projet¶

Developper une application Python orientee objet qui aide les voyageurs a planifier leur sejour en collectant automatiquement des donnees depuis des APIs gratuites.

Fonctionnalites principales :¶

  • Collecte d'informations sur une ville (coordonnees, description)
  • Recuperation de la meteo
  • Recherche d'hotels, restaurants et attractions touristiques
  • Generation d'un programme jour par jour
  • Estimation du budget total du voyage
  • Comparaison de plusieurs destinations
  • Visualisation sur carte interactive

# Creation du programme
programme = voyage_paris.creer_programme()

print("="*60)
print("PROGRAMME DU VOYAGE")
print("="*60)

# Pour chaque jour, on assigne des POIs
for jour in programme.jours:
    print(f"\nJOUR {jour.numero} - {jour.date}")
    print("-" * 60)
    
    # Assigner un hotel
    if hotels_paris:
        jour.hotel = hotels_paris[0]
        print(f"Hotel : {jour.hotel.nom} ({jour.hotel.stars} etoiles - {jour.hotel.prix_estime} euros/nuit)")
    
    # Assigner 3 restaurants pour la journee (petit-dej, dejeuner, diner)
    print(f"\nRestaurants :")
    for i in range(min(3, len(restaurants_paris))):
        idx = (i + (jour.numero-1)*3) % len(restaurants_paris)
        jour.ajouter_restaurant(restaurants_paris[idx])
        resto = jour.restaurants[i]
        print(f"  - {resto.nom} ({resto.type_lieu}) - {resto.prix_estime} euros")
    
    # Assigner 2 attractions pour la journee
    print(f"\nAttractions :")
    for i in range(min(2, len(attractions_paris))):
        idx = (i + (jour.numero-1)*2) % len(attractions_paris)
        jour.ajouter_attraction(attractions_paris[idx])
        attr = jour.attractions[i]
        payant = "Payant" if attr.est_payant() else "Gratuit"
        print(f"  - {attr.nom} ({attr.type_lieu}) - {payant} ({attr.prix_estime} euros)")
    
    # Meteo du jour
    if jour.numero <= len(meteo_paris):
        meteo = meteo_paris[jour.numero - 1]
        print(f"\nMeteo : {meteo.temp_min}C / {meteo.temp_max}C - Pluie : {meteo.precipitation}mm")

print("\n" + "="*60)

Conclusion¶

Travail realise¶

Ce que j'ai realise :¶

  • Collecte automatisee de donnees via 4 APIs Open Sources (Nominatim, Wikipedia, Overpass, Open-Meteo)
  • Conception orientee objet avec 11 classes
  • Traitement et structuration des donnees (CSV, pandas)
  • Visualisations (cartes Folium, graphiques matplotlib/seaborn)
  • Estimation du budget par categorie
  • Programme jour par jour
  • Export des donnees en CSV

Limites :¶

  • Estimation des prix basee sur des moyennes (pas d'API gratuite pour prix reels)
  • Pas d'optimisation des itineraires (calcul de distances)

Ameliorations possibles :¶

  • Integrer une API de prix reels (necessite abonnement payant)
  • Optimiser les trajets jour par jour avec calcul de distances
  • Ajouter plus de criteres de comparaison
  • Systeme de recommandations base sur preferences utilisateur

Retour d'experience¶

Points positifs :¶

  • Les APIs gratuites sont performantes et faciles a utiliser
  • La conception objet facilite l'organisation du code
  • Pandas et matplotlib permettent des visualisations claires
  • Folium cree des cartes interactives attractives

Difficultes rencontrees :¶

  • API Overpass parfois lente ou en timeout
  • Estimation des prix sans API dediee reste approximative
  • Gestion des donnees manquantes (hotels sans etoiles, etc.)

Apprentissages :¶

  • Manipulation d'APIs REST
  • Conception orientee objet en Python
  • Visualisation de donnees avec pandas, matplotlib, seaborn, folium
  • Export et traitement de donnees en CSV

print(" Création du profil Lyon pour comparaison...")

lyon = mv.Ville("Lyon")
lyon.recuperer_coordonnees()
print(f"✓ Lyon : {lyon.latitude}, {lyon.longitude}")

# Transport Lille-Lyon
transport_lyon = mv.Transport("train", lille, lyon)
transport_lyon.calculer_distance_duree()
transport_lyon.estimer_prix()
print(f"✓ Transport : {transport_lyon.distance} km, {transport_lyon.prix_total:.2f} €")

# Collecte rapide des données
hotels_lyon = mv.collecter_hotels(lyon, limite=20)
restaurants_lyon = mv.collecter_restaurants(lyon, limite=30)

# Création du voyage Lyon
voyage_lyon = mv.Voyage(
    utilisateur=utilisateur,
    ville=lyon,
    date_debut="2026-02-10",
    nb_jours=3,
    nb_personnes=2
)

voyage_lyon.set_transport(transport_lyon)

for h in hotels_lyon:
    voyage_lyon.ajouter_hotel(h)

for r in restaurants_lyon:
    voyage_lyon.ajouter_restaurant(r)

# Programme et budget
programme_lyon = voyage_lyon.creer_programme()
for i, jour in enumerate(programme_lyon.jours):
    if i < len(hotels_lyon):
        jour.set_hotel(hotels_lyon[i])

idx = 0
for jour in programme_lyon.jours:
    for _ in range(3):
        if idx < len(restaurants_lyon):
            jour.ajouter_restaurant(restaurants_lyon[idx])
            idx += 1

budget_lyon = voyage_lyon.calculer_budget()

print(f"\n✓ Voyage Lyon créé")
print(f"  Budget total : {budget_lyon.total:.2f} €")
